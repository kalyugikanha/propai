🏠 PropAI Widget — WordPress Embed

Copy the snippet below and paste it into your WordPress site. The widget will automatically appear on every page.

Step 1 — Update your backend URL

Replace https://YOUR-BACKEND.railway.app with your actual Railway deployment URL.
Also update agencyName and agentName to match your brand.

Step 2 — Copy this snippet into WordPress

<!-- PropAI Property Assistant Widget -->
<script>
  window.PropAiConfig = {
    apiBase:     'https://YOUR-BACKEND.railway.app',
    agencyName:  'PropAI Jaipur',
    agentName:   'Priya',
    agentAvatar: '🏠',
    primaryColor: '#7c3aed',
    accentColor:  '#1d4ed8',
    position:    'right'
  };
</script>
<script src="https://YOUR-BACKEND.railway.app/widget/propai-widget.js" defer></script>
<!-- End PropAI Widget -->

Configuration Options

window.PropAiConfig = {
  apiBase:      'https://...',   // REQUIRED — your Railway backend URL
  agencyName:   'Your Agency',  // Shown in widget header
  agentName:    'Priya',        // AI assistant name
  agentAvatar:  '🏠',           // Emoji avatar
  primaryColor: '#7c3aed',      // Primary gradient color (hex)
  accentColor:  '#1d4ed8',      // Accent gradient color (hex)
  position:     'right',        // 'right' or 'left' side of screen
};

Test Locally

During development, set apiBase: 'http://localhost:3000' and open this file in a browser. The widget will appear at the bottom right of this page!
