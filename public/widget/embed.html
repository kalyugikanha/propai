🏠 PropAI — WordPress Embed Guide

Two ways to add PropAI to your WordPress site: a full landing page with property showcase + AI form, or the floating chat widget that appears on every page.

🏡 Option 1 — Full Landing Page (Recommended for Property Leads)

The landing page features a property image slideshow on the left and a 6-field AI search form on the right. Visitors fill details once and get instant property matches + Hot Deals.

Step 1 — Host the landing page

The landing-page.html file must be accessible from a URL. Two options:
  • A (Recommended): Serve it via your backend. The backend already serves static files from frontend/widget/ — so it's available at https://YOUR-BACKEND.railway.app/widget/landing-page.html
  • B: Host it on any static file server / CDN and use that URL below.

Step 2 — Create a Full-Width WordPress Page

In WordPress admin: Pages → Add New → set the page template to "Full Width" (or "No Sidebar"). Then add an HTML block (Elementor HTML widget / WPBakery Raw HTML / Gutenberg Custom HTML) and paste one of the snippets below.

Option A — Elementor / WPBakery HTML Widget (iframe embed)

<!-- PropAI Landing Page — paste into Elementor HTML widget or WPBakery Raw HTML -->
<style>
  #propai-landing-frame {
    width: 100%;
    min-height: 95vh;
    border: none;
    border-radius: 0;
    display: block;
  }
</style>
<iframe
  id="propai-landing-frame"
  src="https://YOUR-BACKEND.railway.app/widget/landing-page.html"
  allow="clipboard-write"
  loading="lazy"
  title="JaipurPropIQ — AI Property Search"
></iframe>

Option B — Gutenberg Custom HTML Block (inline embed)

Alternatively, in a Gutenberg page paste the full HTML content of landing-page.html directly into a Custom HTML block (no iframe needed). This gives better SEO but requires backend URL to be set inside the file first. 
<!-- 1. Open landing-page.html and find the line: -->
const API_BASE = 'http://localhost:3000';

<!-- 2. Change it to your live backend URL: -->
const API_BASE = 'https://YOUR-BACKEND.railway.app';

<!-- 3. Then copy-paste the entire file content into a Gutenberg Custom HTML block. -->

💬 Option 2 — Floating Chat Widget (All Pages)

The floating chat widget appears on every page of your WordPress site as a button at the bottom-right corner. It runs the AI property assistant through a step-by-step conversation flow.

Step 1 — Update your backend URL

Replace https://YOUR-BACKEND.railway.app with your actual Railway deployment URL.
Also update agencyName and agentName to match your brand.

Step 2 — Copy this snippet into WordPress

<!-- PropAI Property Assistant Widget -->
<script>
  window.PropAiConfig = {
    apiBase:     'https://YOUR-BACKEND.railway.app',
    agencyName:  'PropAI Jaipur',
    agentName:   'Priya',
    agentAvatar: '🏠',
    primaryColor: '#7c3aed',
    accentColor:  '#1d4ed8',
    position:    'right'
  };
</script>
<script src="https://YOUR-BACKEND.railway.app/widget/propai-widget.js" defer></script>
<!-- End PropAI Widget -->

Configuration Options

window.PropAiConfig = {
  apiBase:      'https://...',   // REQUIRED — your Railway backend URL
  agencyName:   'Your Agency',  // Shown in widget header
  agentName:    'Priya',        // AI assistant name
  agentAvatar:  '🏠',           // Emoji avatar
  primaryColor: '#7c3aed',      // Primary gradient color (hex)
  accentColor:  '#1d4ed8',      // Accent gradient color (hex)
  position:     'right',        // 'right' or 'left' side of screen
};

Test Locally

During development, set apiBase: 'http://localhost:3000' and open this file in a browser. The widget will appear at the bottom right of this page!
